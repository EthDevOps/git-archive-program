Ethereum Git Archive Program

About the Program

This program aims to preserve a copy of all ethereum relevant git repositories.

The copies are held at a read-only Gitea instance at git.ethquokkaops.io In addition this Gitea instance is backed up every day to off-site EF systems.

Submit new Repo

If you would like to propose a repository for inclusion, please submit an issue to the Git-Sync project repo: EthDevOps/git-archive-program.

When you submit the issue, please specify the following information:

  1. Project repo URL
  2. Title and description
  3. Maintainers and/or sync contact email address
  4. Explanation if it is not immediately clear that the repo is Ethereum-related
  5. Anything especially relevant or timely. For example: Is this repo about to be removed?
